# MX-Cutting-Up-Score

A standalone high-performance scoring script for street racing and "cutting up" in traffic.

## Features
* **Brand:** MX Project
* **Visuals:** Clean white score with blue HUD accents.
* **Logic:** Anti-farm system (score only once per car until 50m distance).
* **Reset:** Instant score wipe upon collision or stopping.
* **Timer:** Integrated countdown session timer.

## Installation
1. Drop the `mx-cutting-up` folder into your resources.
2. Add `ensure mx-cutting-up` to your `server.cfg`.
3. Use command `/nohesi` in-game to toggle.

## Requirements
* `ox_lib` (for notifications)
